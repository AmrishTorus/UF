# Torus Tutorial Intro

Let's explore **Torus in less than 5 minutes** and see how it simplifies application development.

## Getting Started

Get started by **creating a new Torus client**.

Or **experience Torus instantly** with **[Torus](https://app4nx.gsstvl.com/register)**.

### What you'll need

- [Node.js](https://nodejs.org/en/download/) version 18.0 or above:
  - When installing Node.js, ensure you check all dependencies for a smooth setup.
- A registered Torus account to access the platform's powerful Low-Code capabilities.

## Create a New Torus Application

Generate a new Torus project using the **Torus CLI**.

The following command initializes a new project with predefined templates:

```bash
npx torus-cli create my-app
